# Zenlok Flora 🌱

A Full Stack E-Commerce Web Application for selling plants, flowers, vases, and fertilizers.

## Features

- User Registration & Login
- Product Catalog
- Shopping Cart
- Checkout System
- Order Tracking
- Admin Dashboard
- Product Management
- Order Management

## Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
